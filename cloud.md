РюЁ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃўрЃюрЃЊрЃљ Рђћ рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃњрЃўрЃгрЃћрЃа рЃАрЃарЃБрЃџрЃљрЃЊ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃБрЃџ, рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЕрЃљрЃЏрЃЮрЃДрЃљрЃџрЃўрЃЉрЃћрЃЉрЃБрЃџ
**`cloud.md`** рЃцрЃљрЃўрЃџрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЕрЃљрЃАрЃЋрЃљ рЃерЃћрЃюрЃА `~/hyBackup/` рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў.

рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃљрЃдрЃгрЃћрЃарЃА **Drive рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљрЃА, `rclone` рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃА, рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ/рЃЕрЃљрЃЏрЃЮрЃбрЃљрЃюрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃћрЃЉрЃА, рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃА**

## ­ЪЊў рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

**hyBackup** рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃерЃћрЃюрЃА рЃЉрЃћрЃЦрЃљрЃцрЃћрЃЉрЃА рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮ (`~/.backup/`),
рЃљрЃарЃљрЃЏрЃћрЃЊ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЮ рЃЊрЃљ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЮ **Google Drive**-рЃЊрЃљрЃю рЃћрЃарЃЌрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЌ.

рЃћрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА CLI рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃќрЃћ Рђћ **`rclone`**,
рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА cloud storage-рЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљрЃА.

## РџЎ№ИЈ 1№ИЈРЃБ rclone рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ (Manjaro):

```bash
sudo pacman -S rclone
```

### рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ:

рЃњрЃљрЃБрЃерЃЋрЃў:

```bash
rclone config
```

рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃўрЃ░рЃДрЃћрЃЋрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃА:

1№ИЈРЃБ рЃљрЃўрЃарЃЕрЃўрЃћ **`n`** (new remote)
2№ИЈРЃБ рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃАрЃљрЃ«рЃћрЃџрЃў:

```

```

3№ИЈРЃБ рЃљрЃўрЃарЃЕрЃўрЃћ рЃбрЃўрЃърЃў:


рЃљрЃю рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа:

```

```

4№ИЈРЃБ рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃю рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **Enter** (рЃЊрЃћрЃцрЃЮрЃџрЃбрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў)
5№ИЈРЃБ рЃарЃЮрЃфрЃљ рЃњрЃЎрЃўрЃЌрЃ«рЃљрЃЋрЃА `Use auto config?` Рєњ рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **y**
6№ИЈРЃБ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃЊрЃљрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћ
7№ИЈРЃБ рЃЊрЃљрЃЉрЃарЃБрЃюрЃЊрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃЊрЃљ рЃљрЃўрЃарЃЕрЃўрЃћ `y` Рєњ РђюThis is OKРђЮ
8№ИЈРЃБ рЃЉрЃЮрЃџрЃЮрЃА `q` Рєњ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ.

рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ:

```bash
rclone listremotes
```

­ЪЉЅ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА:

## ­ЪЌѓ№ИЈ 2№ИЈРЃБ Drive рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
